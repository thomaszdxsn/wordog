<RepostCard
    title="Thoughts on Foreign Keys"
    link="https://github.com/github/gh-ost/issues/331#issuecomment-266027731"
    tags={["外键", "GitHub"]}
    estimateMin={5}
>

GitHub 工程师回答不用外键的主要理由:

- 外键让你的 app 完全依赖数据库来维护完整性。
- 外键有性能隐患，每一次 insert / delete 都有额外的开销
- 外键让在线 schema 迁移变的很困难

</RepostCard>
